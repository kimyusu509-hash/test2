# 📝 마크다운(Markdown) 문법 총정리 치트시트

---

## 1. 제목 (Headings)

```markdown
# 제목 1 (H1)
## 제목 2 (H2)
### 제목 3 (H3)
#### 제목 4 (H4)
##### 제목 5 (H5)
###### 제목 6 (H6)
```

> ✅ `#` 뒤에 반드시 **공백 한 칸**을 넣어야 합니다.

---

## 2. 텍스트 스타일 (Text Styling)

| 스타일 | 문법 | 결과 |
|--------|------|------|
| 굵게 (Bold) | `**굵은 글씨**` | **굵은 글씨** |
| 기울임 (Italic) | `*기울인 글씨*` | *기울인 글씨* |
| 굵게+기울임 | `***굵고 기울인 글씨***` | ***굵고 기울인 글씨*** |
| 취소선 | `~~취소선~~` | ~~취소선~~ |
| 인라인 코드 | `` `코드` `` | `코드` |
| 하이라이트 | `==하이라이트==` | ==하이라이트== |
| 위첨자 | `X^2^` | X^2^ |
| 아래첨자 | `H~2~O` | H~2~O |

## 3. 줄바꿈과 단락

```markdown
줄 끝에 공백 두 칸을 넣으면  
줄바꿈이 됩니다.

빈 줄 하나를 넣으면 새 단락이 시작됩니다.
```

## 4. 목록 (Lists)

### 순서 없는 목록

```markdown
- 항목 1
- 항목 2
  - 하위 항목 2-1
  - 하위 항목 2-2
    - 더 깊은 항목
```

### 순서 있는 목록

```markdown
1. 첫 번째
2. 두 번째
3. 세 번째
   1. 하위 항목
   2. 하위 항목
```

### 체크박스 (Task List)

```markdown
- [x] 완료된 항목
- [ ] 미완료 항목
- [ ] 또 다른 할 일
```

- [x] 완료된 항목
- [ ] 미완료 항목
- [ ] 또 다른 할 일

## 5. 인용문 (Blockquote)

```markdown
> 일반 인용문입니다.

> 중첩 인용문도 가능합니다.
>> 이렇게 `>`를 추가하면 됩니다.
>>> 세 단계까지도 가능합니다.
```

> 일반 인용문입니다.

> 중첩 인용문도 가능합니다.
>> 이렇게 `>`를 추가하면 됩니다.
>>> 세 단계까지도 가능합니다.

## 6. 코드 블록 (Code Block)

### 인라인 코드

```markdown
`console.log("Hello")`를 실행하세요.
```

### 코드 블록 (언어 지정)

````markdown
```python
def hello():
    print("안녕하세요!")
    return True
```
````

```python
def hello():
    print("안녕하세요!")
    return True
```

> 지원 언어 예시: `python`, `javascript`, `java`, `c`, `html`, `css`, `bash`, `json`, `markdown` 등

## 7. 링크 (Links)

```markdown
[표시 텍스트](https://www.example.com)

[툴팁 있는 링크](https://www.example.com "마우스를 올려보세요")

자동 링크: <https://www.example.com>

참조 링크: [구글][1]
[1]: https://www.google.com
```

---

## 8. 이미지 (Images)

```markdown
![대체 텍스트](이미지URL)

![고양이 사진](https://example.com/cat.jpg "귀여운 고양이")

[![클릭 가능한 이미지](이미지URL)](링크URL)
```

> 💡 이미지 크기 조절은 표준 마크다운에서 지원하지 않지만, HTML을 사용할 수 있습니다:
> `<img src="이미지URL" width="300">`

---

## 9. 표 (Tables)

```markdown
| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
|:----------|:----------:|----------:|
| 내용 1    | 내용 2     | 내용 3    |
| 내용 4    | 내용 5     | 내용 6    |
```

| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
|:----------|:----------:|----------:|
| 내용 1    | 내용 2     | 내용 3    |
| 내용 4    | 내용 5     | 내용 6    |

> `:---` 왼쪽 정렬 / `:---:` 가운데 정렬 / `---:` 오른쪽 정렬

- 코드도 넣을 수 있습니다
